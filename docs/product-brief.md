# Delegated Authority POC

Delegated Authority is a mobile front-end prototype for supervising agent-generated communications.

It demonstrates:

Policy → Behavior → Exceptions → Human judgment

This is not a messaging app or full production product. It is a focused React prototype using static data and simulated behavior.

## Core screens

1. Control Surface
- Policy stance: Strict / Balanced / Autonomous
- Outcome card showing system behavior
- Escalation queue filtered by selected policy

2. Escalation Detail
- Proposed action
- Intent
- Risks
- Suggested actions
- Editable draft message
- Actions: Approve & Send, Edit, Hold

Supporting overlays:
- Policy detail
- Conversation context
- Edit message modal

## Scope boundaries

No backend.
No authentication.
No real message sending.
No notification system.
No external data.
No full messaging interface.
No additional product flows unless required for clarity.
